Overview

Cat Facts API is an ad free, no cost API that sends cat facts via text messages everyday to subscribed recipients. It has the capability to respond to text messages from the recipients with some more interesting cat facts. Following are the other capabilities of the Cat Facts API:

  • Add and view personal list of fact recipients
  • View the catversation between Catbot and your recipients
  • Submit interesting facts
  • Copy-and-paste cat facts using Get Facts
  • Access the admin console panel for managing the app

    • Anybody can signup on the Cat Facts website and perform the above listed activities.

    Getting Started

    Currently, viewing or posting user related data requires authentication. This can be done only by logging in to the website.

    How to make a request?

    Log into the application and click in the lower-right corner to view an interesting cat fact.

    Authentication and Authorization

    This API does not require any authentication except for making user related requests. Please signup and log in to the website to be authorized.

    Rate Limits

    As this app is running on a free Heroku dyno, the response time may be a few seconds long on the first request. Subsequent requests will process as intended.

    Status and Error Codes

    Status Code

    Code Status
    200 Successful request and response
    400 Server could not process your request
    403 Forbidden request
    404 Requested resource was not found
    500 Unexpected server error encountered

    Error Types

    Type Description
    params_invalid Parameters were not valid
    api_error Internal API error
    rate_limit The request has been rate limited
